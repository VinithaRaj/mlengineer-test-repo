Describe audit logging.

Audit logging is the process of keeping track of system activites by user, data, and any changes for the sake of traceability and security. The different ways in which audit logging is performed is by:
i. User authentication - logging user authentication times
ii. User activity - logging queries made by user and resources accessed
iii. Retrieval logs - logging the queries made and the documents accessed for that query, and the outputs produced
iv. Security logs - logging succesful and failed login attempts to detect any breach of privacy
